I am a huge fan of open ended challenges that use easy to find things from around the house. Set everything out, explain the concept and leave them alone for half an hour while you have a sit down and a cup of tea. The competitive element encourages creativity, and there is a real sense of accomplishment at the end. You might be surprised by some of the solutions they come up with.
  
## The Challenge: How Many Blocks Can a Paper Bridge Support?

On the face of it, this is a tricky one. Build a bridge that can support the most amount of blocks using only a few sheets of paper. In fact the list of things you need is limited to two, so if you have the blocks already this one will cost a total of about 5p:

* Some building blocks. [Something like this](https://www.amazon.co.uk/gp/product/B002J0QGD4/ref=as_li_tl?ie=UTF8&camp=1634&creative=6738&creativeASIN=B002J0QGD4&linkCode=as2&tag=howtodad-21&linkId=823d685d6de9e0098875afb34c29958a) is perfect
* A few sheets of paper trimmed to the same width as the blocks

![Ingredient List](/img/posts/bridge1.jpg)

The rules are simple: the two large blocks are a set distance apart, and you can do whatever you like with the paper as long as it does not touch the table in between the large blocks.

## Start Basic

Put the two blocks a set distance apart, and then place a few of the trimmed sheets of paper on top. This basic bridge probably will not be able to support any blocks:

![Basic bridge](/img/posts/bridge2.jpg)

## Try Different Styles of Bridges

The arch beam looks great and can start to support some of the other blocks

![Basic bridge](/img/posts/bridge3.jpg)

When we did this we found the winner was this type of bridge, which is kind of like a [truss bridge](https://www.garrettsbridges.com/design/trussdesign/). Folding the paper in this way creates triangles which distributes the weight evenly.

![Basic bridge](/img/posts/bridge4.jpg)
