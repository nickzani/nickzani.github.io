I am a huge fan of open ended challenges that use easy to find things from around the house. The competitive element encourages creativity, and there is a real sense of accomplishment at the end. 
  
## The Challenge: How Many Blocks Can a Paper Bridge Support?

On the face of it, this is a tricky one. Build a bridge that can support blocks using only a few sheets of paper. In fact list of things you need is limited to two, so if you have the blocks already this one will cost a total of about 5p:

* Some building blocks. [Something like this](https://www.amazon.co.uk/gp/product/B002J0QGD4/ref=as_li_tl?ie=UTF8&camp=1634&creative=6738&creativeASIN=B002J0QGD4&linkCode=as2&tag=howtodad-21&linkId=823d685d6de9e0098875afb34c29958a) is perfect
* A few sheets of paper trimmed to the same width as the blocks

![Ingredient List](/img/posts/bridge1.jpg)

## Start Basic

Put the two blocks a set distance apart, and then place a few of the trimmed sheets of paper on top. This basic bridge probably will not be able to support any blocks:

![Basic bridge](/img/posts/bridge2.jpg)
